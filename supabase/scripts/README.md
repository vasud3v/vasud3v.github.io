# Supabase Scripts

This directory contains utility SQL scripts for database setup, fixes, and maintenance.

## Setup Scripts

- **SIMPLE_SETUP.sql** - Basic database setup
- **SETUP_CATEGORIES.sql** - Category setup
- **CREATE_MISSING_TABLES.sql** - Create missing database tables
- **CREATE_ADMIN_USER_AND_CONTENT.sql** - Admin user and initial content

## Data Creation Scripts

- **CREATE_POSTS.sql** - Create posts
- **CREATE_TOPICS.sql** - Create topics
- **CREATE_THREADS_FOR_TOPICS.sql** - Create threads for topics
- **CREATE_SINGLE_CATEGORY.sql** - Create a single category

## Fix Scripts

- **COMPLETE_FIX.sql** - Complete database fix
- **FIX_RLS_POLICIES.sql** - Fix RLS policies
- **FIX_CATEGORIES_RLS.sql** - Fix categories RLS
- **FIX_THREADS_RLS.sql** - Fix threads RLS
- **FIX_POLLS_RLS.sql** - Fix polls RLS
- **FIX_REPUTATION_EVENTS.sql** - Fix reputation events

## Update Scripts

- **UPDATE_ALL_POSTS_DETAILED.sql** - Detailed posts update
- **UPDATE_RULES_POSTS.sql** - Update rules posts

## Check Scripts

- **CHECK_RLS.sql** - Check RLS policies
- **CHECK_THREADS.sql** - Check threads
